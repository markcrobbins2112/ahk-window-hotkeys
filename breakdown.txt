================================================================================
FULLY INTEGRATED BULK & FOUR-SIDED SYMMETRICAL OPERATIONS
================================================================================
Core bulk operations and four-sided symmetrical commands have been fully resolved and mapped:

1. Bulk Symmetrical Window Restorations (Bulk-Restore):
   - RestoreAll (Win + [)             - Symmetrically restores all desktop windows back to normal size.
   - RestoreAllMinimized (Win+Ctrl+[) - Selectively restores minimized windows back to normal size.
   - RestoreAllMaximized              - Selectively restores maximized windows back to normal size.
   - MaximizeAll (Win + ])            - Symmetrically maximizes all open window handles.
   - MaximizeAllRestored              - Selectively maximizes restored windows back to full size.
   - MaximizeAllMinimized             - Selectively maximizes minimized windows back to full size.

2. Four-Sided Symmetrical Grid-Snapping & Border Stretches (Win + Modifiers + Numpad 5):
   - StretchToGridAll (Win+Numpad5)   - Symmetrically snaps and stretches all four borders outward to matching grid lines.
   - PullToGridAll (Win+Ctrl+Alt+Shift+Numpad5) - Symmetrically snaps and pulls all four borders inward to matching grid lines.
   - AddAll (Win+Alt+Shift+Numpad5)    - Symmetrically grows all four window borders outward.
   - SubtractAll (Win+Ctrl+Alt+Numpad5)- Symmetrically contracts all four window borders inward.
   - StretchAll (Win+Shift+Numpad5)   - Symmetrically extends active window boundaries to monitor limits.
   - TrimAll (Win+Alt+Numpad5)        - Symmetrically trims active window borders inward.
